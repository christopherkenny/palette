# Browse a palette on coolors.co

Browse a palette on coolors.co

## Usage

``` r
palette_browse(x)

palette_browse_url(x)
```

## Arguments

- x:

  A palette

## Value

A URL to the palette

## Examples

``` r
palette_browse(roygbiv)
#> [1] "https://coolors.co/FF4444-FFAE4D-FFFF60-50FF50-3939FA-AB3FFB-F68BF6"
```
